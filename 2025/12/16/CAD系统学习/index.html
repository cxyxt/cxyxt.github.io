CAD系统学习

第二章 基本图形绘制及练习

2-1 直线命令与动态输入

  • 图标：📏
  • 英文命令：LINE
  • 快捷键：L
  • 操作说明：创建一系列连续的直线段，每条线段可单独编辑。
  • 关键操作：
    • 放弃上一步：U
    • 闭合图元：C
    • 退出绘制：空格/回车、ESC
  • 动态输入 (F12)：
    • 先输入长度，按 TAB 键，再输入角度（角度值不能用空格确认）。
    • 水平向右为0度，角度始终显示正数。

2-2 绘图辅助工具-正交与栅格捕捉

  • 正交 (F8)：限制光标在水平或垂直方向移动。
  • 图形栅格 (F7)：显示栅格点阵，间距可自定义。
  • 栅格捕捉 (F9)：使光标按设定间距移动，通常与栅格间距一致。

2-3 绘图辅助工具-极轴追踪

  • 快捷键：F10
  • 作用：沿设定角度（增量角）的倍数方向进行追踪。
  • 增量角设置：取图形中所有角度的最大公约数。
  • 注意：正交模式与极轴追踪不能同时开启。

2-4 极轴追踪实例练习

  • （练习内容：使用增量角10°进行绘图）

2-5 绝对坐标绘制直线

  • 概念：基于UCS原点 (0,0) 的坐标。
  • 输入格式：#X,Y。开启动态输入时，第一点可省略 #。

2-6 相对坐标绘制直线

  • 概念：相对于前一个输入点的坐标。
  • 输入格式：@X,Y。

2-7 极坐标绘制直线

  • 概念：基于与前一点的距离和角度。
  • 输入格式：@距离<角度。
  • 角度方向：逆时针为正，顺时针为负。

2-8 绘图辅助工具-对象捕捉

4-1 单行文字的创建与编辑

  • 命令：TEXT，快捷键 DT
  • 作用：创建一行或多行独立文本对象，适用于简短文字标注。
  • 特点：操作简单、位置精确、编辑灵活。
  • 操作步骤：
    1. 激活命令。
    2. 选择对正方式。
    3. 指定文字起点。
    4. 指定文字高度。
    5. 指定旋转角度。
    6. 输入文本，按两次回车完成。
  • 编辑方法：
    • 移动插入点调整位置。
    • 双击文字修改内容。
    • 选中后按 Ctrl+1 打开特性面板，修改颜色、大小、对正、倾斜等。

4-2 文字的15种对正方式

  • 概念：指定文字插入点（对正点）相对于文字边框的位置。
  • 对正类型：左上(TL)、中上(TC)、右上(TR)、左中(ML)、正中(MC)、右中(MR)、左对齐(L)、居中(C)、右对齐(R)、左下(BL)、中下(BC)、右下(BR)、中间(M)、对齐(A)、布满(F)。
  • 参照线说明：
    • 顶线：大写字母和中文最高点，TL/TC/TR的参照。
    • 基线：大写字母和中文下边线，L/C/R的参照。
    • 中线：基线与顶线的中心线，ML/MC/MR的参照。
    • 底线：小写字母最低点，BL/BC/BR的参照。
  • 特殊对正：
    • **对齐(A)**：文字在两点间平均分布，宽高比不变。
    • **布满(F)**：文字在两点间平均分布，高度不变。
    • **中间(M)**：插入点在顶线与底线中间（几何中心）。

4-3 文字样式的参数

  • 命令：STYLE，快捷键 ST
  • 作用：控制文字外观（字体、字号、效果等）。
  • 关键参数：
    • 字体：系统字体(.ttf)或CAD字体(.shx)。使用.shx字体输入汉字需勾选“使用大字体”。
    • 高度：设置默认文字高度。设为0则在创建文字时单独指定。
    • 效果：颠倒、反向、垂直、宽度因子、倾斜角度。
  • **机械制图标准 (GB/T14665-2012)**：
    • 字体：正体输出。
    • 常用字体：gbenor.shx + gbcbig.shx（正体）；gbeito.shx + gbcbig.shx（斜体）；gdt.shx（形位公差符号）。
  • 外来字体安装：复制.ttf文件到系统“字体”文件夹，重启CAD。

4-4 自定义文字样式

  • 创建步骤：
    1. 输入 ST 打开文字样式窗口。
    2. 点击“新建”，输入样式名。
    3. 定义字体、高度、宽度因子等效果。
    4. 点击“应用”，新样式自动置为当前。
  • 重命名与删除：在样式列表中右键选择。

4-5 多行文字的创建

  • 命令：MTEXT，快捷键 MT 或 T
  • 作用：创建多行段落文字，适用于复杂注释（技术说明、材料清单等）。
  • 特点：格式丰富（加粗、倾斜、颜色、背景等）、支持段落控制（对齐、行距、编号）。
  • 操作步骤：

    • 多行转单行：使用分解命令 X。

4-6 特殊符号的输入方法

  • 方法一（代码）：%%d→°、%%p→±、%%c→Φ。

第五章 尺寸标注与公差标注

5-1 尺寸标注的组成要素及基本要求

  • 组成要素：
    • 标注文字：表达测量值，可包含前缀、后缀、公差。
    • 尺寸线：指示标注方向和范围。
    • 箭头（尺寸终端）：显示尺寸线两端。
    • 尺寸界线（投影线）：从部件延伸到尺寸线。
    • 中心标记：用于指示圆或对称中心。
  • 国家标准 (GB) 基本规定：
    • 尺寸线、尺寸界线用细实线。
    • 尺寸线应平行于所标线段。轮廓线、中心线等不可作为尺寸线。
    • 同一图样尺寸终端形式应一致。
    • 尺寸数字一般注在尺寸线上方或中断处，字号一致。
    • 线性尺寸数字方向：水平朝上、竖直朝左、倾斜保持朝上趋势。
    • 角度数字一律水平书写。
    • 任何图线不得通过尺寸数字，必要时图线应断开。
  • 注意事项：
    • 零件真实大小以标注数值为准，与图形比例无关。
    • 默认单位毫米(mm)，否则需注明。
    • 标注为最后完工尺寸。
    • 每一尺寸一般只标注一次，标注在最清晰反映该结构的视图上。
    • 功能尺寸直接注出；同一要素尺寸尽量集中标注。
    • 尽可能使用符号或缩写词。

5-2 线性标注与选项功能

  • 命令：DIMLINEAR，快捷键 DLI
  • 作用：标注两点间的水平或垂直距离。
  • 操作步骤：激活命令 → 指定第一条和第二条尺寸界线原点或选择对象 → 指定尺寸线位置。
  • 命令行选项：
    • 多行文字(M)：打开多行文字编辑器手动输入内容。
    • 文字(T)：命令行输入单行文字。
    • 角度(A)：旋转尺寸文字。
    • 水平(H)/垂直(V)：强制标注水平或垂直尺寸。
    • 旋转(R)：旋转尺寸线角度。

5-3 对齐标注及练习

  • 命令：DIMALIGNED，快捷键 DAL
  • 作用：创建平行于所选对象或两点连线的标注（用于倾斜图线）。
  • 操作步骤：激活命令 → 指定尺寸界线原点或选择对象 → 指定尺寸线位置。

5-4 半径标注与直径标注

  • 半径标注：
    • 命令：DIMRADIUS，快捷键 DRA
    • 操作：激活命令 → 选择圆弧或圆 → 指定尺寸线位置。
  • 直径标注：
    • 命令：DIMDIAMETER，快捷键 DDI
    • 操作：激活命令 → 选择圆或圆弧 → 指定尺寸线位置。

5-5 角度标注与角度格式

  • 命令：DIMANGULAR，快捷键 DAN
  • 作用：标注两线夹角或圆弧圆心角。
  • 操作步骤：激活命令 → 选择两条直线或圆弧 → 指定标注弧线位置。
  • 标注大于180°角：指定顶点 → 逆时针选择起点和端点。
  • 角度格式设置：在标注样式管理器 → “主单位”选项卡中设置（十进制/度分秒）。

5-6 弧长标注及练习

  • 命令：DIMARC，快捷键 DAR
  • 作用：标注圆弧或多段线圆弧段的长度，文字前加弧长符号。
  • 操作步骤：激活命令 → 选择弧线段 → 指定标注位置。
  • 选项：
    • 部分(P)：标注部分弧长。
    • 引线(L)：为尺寸文字添加指引线。

5-7 坐标标注与折弯标注

  • 坐标标注：
    • 命令：DIMORDINATE，快捷键 DOR
    • 作用：测量点的X或Y坐标值。
    • 操作：先移动UCS原点 (UCS → O) → 激活命令 → 指定点 → 移动鼠标切换X/Y值 → 指定引线端点。
  • 折弯标注：
    • 命令：DIMJOGGED，快捷键 DJO
    • 作用：当圆弧中心位于布局外时，指定替代中心进行标注。
    • 操作：激活命令 → 选择圆弧或圆 → 指定图示中心位置 → 指定尺寸线位置 → 指定折弯位置。

5-8 创建标注样式与子样式

  • 命令：DIMSTYLE，快捷键 D
  • 作用：创建和管理控制标注外观的样式。
  • 创建步骤：激活命令打开管理器 → “新建” → 输入名称 → 设置参数 → 确定。
  • 子样式：基于某样式创建，仅修改特定标注类型（如角度、半径）的参数，其余继承父样式。
  • 其他功能：
    • 替代：临时修改当前样式参数，更换样式后失效。
    • 比较：比较两个样式的差异。

5-9 标注样式参数详解（上）

  • 线：
    • 尺寸线：颜色、线型、线宽、超出标记、基线间距、隐藏。
    • 尺寸界线：颜色、线型、线宽、隐藏、超出尺寸线、起点偏移量、固定长度。
  • 符号和箭头：
    • 箭头：第一/第二个箭头形状、引线箭头、箭头大小。
    • 圆心标记：无、标记（十字）、直线（中心线）。
    • 折断标注：折断大小。
    • 弧长符号：标注文字的前缀/上方/无。
    • 半径/线性折弯标注：折弯角度、折弯高度因子。

5-10 标注样式参数详解（下）

  • 文字：
    • 外观：文字样式、颜色、填充颜色、高度、分数高度比例、绘制文字边框。
    • 位置：垂直/水平位置、观察方向、从尺寸线偏移。
    • 对齐：水平、与尺寸线对齐、ISO标准。
  • 调整：
    • 调整选项：当空间不足时，调整文字和箭头的位置策略。
    • 文字位置：尺寸线旁边/上方带引线/上方不带引线。
    • 标注特征比例：注释性、缩放到布局、使用全局比例。
    • 优化：手动放置文字、在尺寸界线间绘制尺寸线。
  • 主单位：
    • 线性标注：单位格式、精度、分数格式、小数分隔符、舍入、前缀/后缀。
    • 测量单位比例：比例因子、仅应用到布局标注。
    • 消零：前导/后续零、英尺/英寸消零。
    • 角度标注：单位格式、精度、消零。
  • 换算单位：
    • 用于显示双单位（如公制/英制）。
    • 设置单位格式、精度、换算倍数、舍入、前缀/后缀、消零、位置（主值后/下）。
  • 公差：
    • 公差格式：方式（无/对称/极限偏差等）、精度、上/下偏差、高度比例、垂直位置。
    • 公差对齐：对齐小数分隔符/运算符。
    • 消零：前导/后续零。

  3. 平行尺寸按大小顺序排列（内小外大），间距一致。
  4. 内形、外形尺寸分别注在视图两侧。
  5. 定形尺寸尽量集中注在同一视图。
  6. 尺寸界线一般垂直尺寸线，必要时可倾斜。
  7. 圆角处用细实线延长轮廓，从交点引尺寸界线。
  8. 对称图形画一半时，尺寸线应略超过中心线，仅一端画箭头。
  9. 重要尺寸（如定位尺寸）应从设计基准直接标出。
  10. 考虑工艺基准和加工顺序。
  11. 考虑测量的方便性。
  12. 避免形成封闭尺寸链。

5-28 尺寸公差标注及练习

  • 基本概念：
    • 基本尺寸：设计尺寸。

    • 上/下偏差：极限尺寸减基本尺寸所得代数差。
    • 公差：允许的变动量。
  • 标注方式：
    1. 直接标注偏差数值（如 30±0.01）。
    2. 标注公差带代号（如 Φ30K7, Φ30f8）。
    3. 在技术要求中统一说明（如“未注公差按GB/T 1804-m”）。

6-1 机械制图关于图线的规定

  • 参照标准：GB/T 4457.4-2002 (图线)，GB/T 14665-2012 (CAD制图规则)。
  • 线宽：采用粗、细两种线宽，比例 2:1。组别选择依据图样类型、尺寸、比例等。
  • **线型与颜色 (屏幕显示推荐)**：
    • 粗实线：白色。用途：可见轮廓线、相贯线等。
    • 细实线：绿色。用途：尺寸线、尺寸界线、剖面线、引出线等。
    • 波浪线/双折线：绿色。用途：边界线、断裂处边界线。
    • 细虚线：黄色。用途：不可见轮廓线。
    • 粗虚线：白色。用途：允许表面处理的表示线。
    • 细点画线：红色。用途：轴线、对称中心线等。
    • 粗点画线：棕色。用途：限定范围表示线。
    • 细双点画线：洋红色。用途：相邻零件轮廓线、假想轮廓线等。

  • 线型管理：
    • 加载：打开线型管理器 → “加载” → 选择线型。
    • 删除：在线型管理器中选择线型 → “删除”。

6-2 自定义线型的两种方法

  • 方法一：修改现有线型文件
    1. 找到CAD线型文件（如 acadiso.lin），用记事本打开。
    2. 复制现有线型定义，修改名称和参数（如将 U=0.0 改为 R=0.0 可反转）。
    3. 保存文件，在CAD中重新加载线型。
  • 方法二：通过绘制图形创建
    1. 按1:1比例绘制线型图形（使用单行文字，中文需勾选大字体）。
    2. 输入命令 MKLTYPE (MK)。
    3. 设置保存路径、文件名，定义线型名称和描述。
    4. 指定起点和终点，选择绘制的对象。
    5. 注意：若实线长度异常，需编辑.lin文件调整参数后重新加载。

6-3 创建新图层、机械制图样板

  • 图层概念：类似透明胶片，用于分类管理不同属性的对象。
  • 创建图层：
    • 命令：LAYER，快捷键 LA。
    • 步骤：打开图层特性管理器 → “新建图层” → 重命名 → 定义颜色、线宽、线型、透明度等。
    • 建议：为标注、剖面线、细实线等分别创建图层，便于管理。
  • 创建机械制图样板文件步骤：
    1. 打开含所需样式（文字、标注、引线、块）的DWG文件。
    2. 删除所有图形对象。
    3. 打开图层管理器，删除旧图层。
    4. 从其他文件复制已按规范设置的图层信息并粘贴。
    5. 再次删除所有图形对象。
    6. “另存为” → 文件类型选择“图形样板(*.dwt)”，命名（如“机械制图2”）。

    • LAYLCK：锁定选定对象所在图层。

第七章 参数化绘图

7-1 CAD 参数化绘图的优势

CAD 参数化绘图是一种设计方法，它通过定义尺寸和几何约束来创建可以轻松修改的设计。在参数化绘图中，当一个参数（如长度、宽度或角度）改变时，整个设计会自动更新以保持一致性。这种方法提高了设计效率，并减少了因手动调整而导致的错误。

参数化绘图优势：

  1. 简化绘图步骤
  2. 防止图元变形
  3. 在设计中随时修改尺寸
  4. 绘制一些难画的图形
  5. 尺寸支持运算
  6. 使用关系式关联尺寸

7-2 几何约束与标注约束

参数化绘图是通过添加约束来实现的。约束可以分为两类：几何约束和标注约束。

  • 几何约束：定义图形元素之间的相对位置关系，比如平行、垂直、相切、同心等。使用几何约束，可以确保即使移动或调整图形时，相关的几何关系也能保持不变。
  • 标注约束：指定图形元素的确切大小，如长度、角度、半径等。通过标注约束，可以直接控制图形的具体尺寸，当改变这些约束值时，整个图形会自动更新以满足新的条件。

在实际使用中，几何约束和标注约束通常是混合使用的，优先使用几何约束来确定图形的基本形状和位置，然后再使用标注约束来进一步细化和调整图形的大小。

当图形处在完全约束状态，既不会变形，也不能移动。

几何约束与捕捉的区别：

  • 捕捉是先期约束，是临时的，是不固定的。
  • 几何约束是后期约束，是永久的，是恒定的。

标注约束与尺寸标注的区别：

  • 尺寸标注是对图形的注释，不能改变图形的大小。
  • 标注约束是对图形的驱动，可以改变图形的大小。

7-3 几何约束的类型与操作方法—重合/共线/同心/水平/竖直/固定

几何约束的类型包括：重合、共线、同心、水平、竖直、相等、平行、垂直、相切、对称、平滑、固定。

  • 重合：约束两个点使其重合，或约束一个点位于对象上。

    • 先选择的点或对象作为参照基准，保持不动。
    • 多个：先选择对象，再选择多个点，与对象重合。
    • 自动约束：当两个点已经相连，可使用自动约束。

  • 共线：约束两个点或线段在同一直线上。

    • 适用于直线段、多段线线段、文字、椭圆（弧）轴端点。
    • 多个：将多个对象约束在一条直线上。

  • 同心：约束两个对象的圆心点重合。

    • 适用于圆、圆弧、椭圆、椭圆弧。

  • 水平：约束一条直线或者两个点，使其与当前的 UCS 的 X 轴平行。

  • 竖直：约束一条直线或者两个点，使其与当前的 UCS 的 Y 轴平行。

  • 固定：约束点或对象的位置，使其不能移动。

7-4 几何约束的类型与操作方法—相等/平行/垂直/相切/对称/平滑

  • 相等：约束两条直线或者多段线使其具有相同的长度，或约束圆和圆弧使其具有相同的半径值。

    • 选择的第一个对象为参照尺寸。
    • 多个：将多个对象约束为相等。

  • 平行：约束两条直线、文字、椭圆，使其具有相同的角度。

    • 选择的一个对象为参照，保持不动。
    • 直线：第二个对象靠近选择点的端点作为旋转基点。
    • 椭圆：圆心点作为旋转基点。
    • 文字：对正点作为旋转基点。

  • 垂直：约束两条直线、文字、椭圆，使其夹角始终保持90°。

    • 选择的第一个对象为参照，保持不动。
    • 直线：第二个对象靠近选择点的端点作为旋转基点。
    • 椭圆：圆心点作为旋转基点。

第八章 三维实体建模与编辑

8-1 三维建模界面及选项设置

CAD 三维实体建模是一种使用计算机软件创建和编辑虚拟三维实体的过程。

三维实体建模的内容：

  1. 用户坐标系（UCS）的建立：学会根据需要调整坐标系的位置与方向，以便更高效地绘制和编辑三维模型。
  2. 三维实体的观察：学会使用不同的视角和视觉样式来观察三维模型，这有助于更好地理解模型的空间关系。
  3. 实体建模：学会长方体、圆柱体等基本体的创建，以及使用拉伸、旋转、放样、扫描建模。
  4. 布尔运算：布尔运算包括并集、差集和交集，这些运算可以用来组合、分割和修剪实体。
  5. 实体编辑：熟悉如何使用 CAD 软件提供的编辑命令，如倒角、圆角、移动、旋转、阵列、镜像、面编辑等，来修改实体的位置、方向、数量和细节。

三维建模界面

点击工作空间，切换到三维基础或三维建模。
功能区是三维建模的选项卡和工具面板。
绘图区的左上角是视口控件，右上角是视角立方体，右下角是导航栏，左下角是坐标系。

视口控件有三个选项，分别是显示选项、视图控件、视觉样式控件，显示选项用来控制视角立方体、全导航控制盘、导航栏的显示与隐藏以及视口的配置。

WCS（世界坐标系）与 UCS（用户坐标系）

  • WCS（世界坐标系）：一个固定的、默认的坐标系统，其原点和坐标轴方向不会改变。
  • UCS（用户坐标系）：一个可移动的坐标系统，允许用户根据绘图需求自定义原点和坐标轴的方向。UCS 的灵活性使得在特定的绘图任务中更加方便，例如在三维模型的某个面上绘制图形或者处理倾斜的图纸。

使用场景和优势：

  • WCS：适用于初始绘图阶段，当绘图的方向与 WCS 一致时，可以直接使用。WCS 提供了一个稳定的参考框架，有助于初学者理解和学习 CAD 软件。
  • UCS：在需要特定视角或在三维空间中工作时更为有用。通过调整 UCS，用户可以更直观地绘制和编辑对象，提高工作效率。例如，在建模时可以将 UCS 放置在模型的侧面或顶面上，以便沿着这些平面进行精确的操作。

切换和控制：可以使用 UCS 命令来激活 UCS 编辑，并通过不同的选项来设置新的原点、旋转 UCS 或将 UCS 与现有对象对齐。也可以使用 WCS 选项将 UCS 重置为 WCS。

绘图区优化

状态栏：添加三维对象捕捉、动态 UCS。
选项：系统-图形性能、三维建模-十字光标。

切换视觉样式通常有两种方式：

  • 方式一：点击绘图区左上角的视觉样式控件，通过弹出的菜单切换视图显示效果。

  • 灰度：将对象以单色模式着色，以产生灰色效果。
  • 勾画：将对象显示为手绘的效果。
  • 线框：用直线和曲线显示对象的边缘，对象的线型和线宽都是不可见的，坐标系按钮显示成三维着色形式。

8-28 实体面的编辑（同步建模）

实体面的编辑也就是同步建模。同步建模融合了直接建模的灵活性和参数化设计的精确控制，可以直接在模型上进行各种操作，无需考虑复杂的特征历史或参数关系，大大提高了设计效率。
主要编辑方法有：拉伸面、倾斜面、移动面、复制面、偏移面、删除面、旋转面、着色面。

拉伸面：英文命令：SOLIDEDIT
按指定的距离或沿某条路径拉伸三维实体的选定平面，拉伸操作可以增加或减少实体的体积。

  • 输入正值可向外侧拉伸面，输入负值可向内侧拉伸面。
  • 倾斜角：将边倒角至面。
  • 路径：沿选定的直线或曲线拉伸面。

倾斜面：英文命令：SOLIDEDIT
将选定的实体面沿着一个预设的轴线倾斜一定的角度。常用于在实体上创建特定角度的倾斜特征，如为了制造方便而设计的拔模角度。

移动面：英文命令：SOLIDEDIT
将三维实体上的面在指定方向上移动指定距离。可以通过移动面来更改对象的位置，在三维空间中重新定位实体面。例如可以移动凸台、孔的位置。

复制面：英文命令：SOLIDEDIT
复制三维实体上的面，从而生成面域或实体。通过面的原始方向和轮廓创建新对象。可将结果用作创建新三维实体的参照。

偏移面：英文命令：SOLIDEDIT
在实体面上创建等距的断面，增加或减少实体厚度。可以修改弧面的半径。

删除面：英文命令：SOLIDEDIT
删除三维实体上的面。使用此选项可删除凸台、孔、圆角和倒角边。

旋转面：英文命令：SOLIDEDIT
绕指定轴线旋转实体面。旋转面常用于创建对称的几何形状或模拟物体的旋转运动。
定义旋转轴的方法：

  • 两点：指定两点来确定旋转轴，轴的正方向由第一个选择点指向第二个选择点。
  • 经过对象的轴：通过图形对象来定义旋转轴。若选择直线，则所选直线即是旋转轴。若选择圆或圆弧，则旋转轴通过圆心且垂直于圆或圆弧所在的平面。
  • 视图：旋转轴垂直于当前视图，并通过拾取点。
  • X 轴、Y 轴、Z 轴：旋转轴平行于 x、y、z 轴，并通过拾取点。旋转轴的正方向与坐标轴的正方向一致。
  • 指定旋转角度：输入正或负的旋转角，旋转角的正方向由右手螺旋法则确定。
  • 参照：该选项允许用户指定旋转的起始参考角和终止参考角，这两个角度的差值就是实际的旋转角，此选项常常用来使表面从当前的位置旋转到另一指定的方位。

着色面：英文命令：SOLIDEDIT
更改三维实体上选定面的颜色。

8-33 三维实体的尺寸标注

CAD 三维尺寸标注的方法

  1. 激活二维捕捉和三维捕捉：以精确的指定尺寸界线的原点。
  2. 调整 UCS（用户坐标系统）：将 UCS 放置在需要添加标注的对象面上，以便尺寸标注与对象的实际表面对齐。
  3. 放置尺寸标注：在“注释”选项卡中选择相应的标注工具，在三维模型上放置尺寸，尺寸文本要保持向上的趋势。

三维标注的注意事项

  1. 避免遮挡：确保标注不会遮挡重要图形元素或其他标注。
  2. 字体一致性：尺寸文字保持统一的样式、大小。
  3. 方向一致性：平行平面的尺寸文字保持一致的方向。
  4. 合理利用空间：优先选择模型周围未使用的空间放置标注。
  5. 考虑观察角度：从主要观察角度可见的位置放置关键标注。
  6. 使用辅助视图：复杂模型中，可创建局部放大视图或剖面视图进行标注。

坐标系的设置是进行精确标注的基础：

  • **面(F)**：将 UCS 动态对齐到三维对象的面。这在标注平面对象时非常有用，可以确保标注与对象表面平行。
  • **对象(OB)**：使用特定对象（如圆弧、圆、标注、线等）来定义新的 UCS。这种方法特别适合在复杂几何形状上进行标注。
  • 3点（3）：通过指定三个点来定义新的 UCS。这种方法可以灵活地创建任意方向的坐标系，适用于非标准几何形状的标注。
  • 视图（V）：以平行于屏幕的平面为 XY 平面，建立新的坐标系。这有助于在进行屏幕截图或演示时保持标注的一致性。

在设置 UCS 时，还需注意以下几点：

  • 坐标轴方向：确保 X、Y 轴的方向与待标注的对象相匹配，避免尺寸文本反向或颠倒。
  • 原点位置：根据标注需求选择合适的原点位置，通常选择在待标注对象的关键点上。
  • Z轴方向：通过指定 Z 轴方向，可以进一步精确控制 UCS 的方位。
