设置标注精确值

UNITS
设置精度：输入 UNITS，在角度精度里设置你需要的小数位数（如 0.000）。

精确测量：输入 MEA 选 角度 或 DIMANGULAR。

选择对象：精确捕捉选择两条直线（或圆弧，或指定三个点）。

查看结果：看命令行或按 F2 看文本窗口，结果就是带小数点的精确角度值。

通过正确设置单位精度并使用测量命令，CAD可以轻松提供你需要的、包含小数点的夹角精确度数。💻

1-2 材料的力学性能指标

材料选型的预备知识和概念
材料的力学性能指标:

  • 强度:抵抗永久变形和断裂的能力。
    载荷的作用方式:拉压弯剪扭。
    相关参数:屈服强度、抗拉强度、抗压强度等。

  • 刚度:抵抗弹性变形的能力，即受力下抵抗弹性变形的能力 针对某个结构或构件而言。
    相关参数:弹性模量、截面和形状。

  • 硬度:衡量材料软硬的程度。
    相关参数:布氏硬度、洛氏硬度、邵氏硬度等

  • 弹性:
    受到外力下，不发生塑性变形的能力。
    相关参数:屈服强度

  • 塑性:材料能忍受塑性变形而不破坏的能力。
    材料在承受最大载荷破坏时，变形的最大百分比。

  • 耐磨性:抵抗摩擦作用的能力。常用硬度值作为衡量材料耐磨性的指标之一，但耐磨还与表面光洁度，使用条件，自润滑性有关

  • 抗疲劳性:机械零件长期反复受到随时间做周期性变化的交变应力时出现的突然断裂的现象称为零件疲劳失效。

    选材料时具体侧重哪方面，选何种材料，要看材料的具体应用场所(关键部位才校核)。

1-3 钢材牌号和种类

  1. 材料牌号的认识

  2. 金属材料的基本概念认知:

  • 有色金属和黑色金属:黑色金属指铁及其合金，如钢、铸铁等。
    黑色金属以外的金属称为有色金属又叫非铁金属，广义的有色金属还包括有色合金。那么比如铜铝金银等包括他们的合金都属于有色金属。
  • 钢的来源:铁矿石(埋在底下的一堆石头)—-进炉子炼成生铁(碳含量在2.11%到4.3%，且杂质非常多，脆且易断裂)—-进一步提纯炼制—-经过一系列步骤才炼得钢。从概念上讲，铁是元素，钢是包含铁和碳以及其余元素的合金，我们平时口头上说的铁其实就是铁碳合金，也就是钢。
  • 钢的定义和分类:含碳量介于0.02%至2.11%之间的铁碳合金。也叫碳钢，非标设计中最常用的金属材料。按照碳钢中含碳量的多少，进一步分为:低碳钢、中碳钢和高碳钢。随着含碳量升高，碳钢的硬度增加、韧性下降，即越脆。
  • 合金钢:为了改善钢的性能，在冶炼时，加入一些合金元素而炼成的钢，比如不锈钢就是一种合金钢。碳钢虽有良好的力学和加工性能，但它存在不易完全淬透，强度不够高，不耐蚀，耐高温，耐磨等特点。”
  • 工具钢:适合制成工具(刀具、量具、模具、耐磨工具等)的各种碳钢(也就是高碳钢)和合金钢。主要特点:高硬度、高耐磨性和抗变形性。
  • 高速钢(HSS):又叫锋钢或白钢，属于工具钢。高速钢的叫法是相对一般工具钢而言的，最主要的特性就是”红硬性”，即高速加工时产生高热的情况下依旧保持高硬度(大于60度)的性能。比如在500度的时候，刀其的硬度仍能保持在60度以上，而普通的工具钢，温度高于200时，硬度便急剧下降。那么具有红硬性也就是允许的切削速度更快，主要就是用在加工刀具上。
  • 硬质合金(钨钢):由难熔金属的硬质化合物和粘结金属通过粉末冶金工艺制成的一种合金材料。主要用在刀具材料上，硬度八九十。加工速度更快，硬度更高更耐磨。500度硬度基本不变，1000度仍有很高硬度。
  • 铸铁:含碳量少于0.02%的叫熟铁，即“纯铁”。含量在2.11%到4.3%的叫“生铁”，属于炼钢原材料，而铸铁的含碳量与熟铁差不太多，但是铸铁是从生铁经过二次加工后的成品。铸铁又分白口铸铁，工业上很少用，主要当炼钢原料用，硬而脆灰口铸铁，铸造性，加工性，耐磨，减震都表现不错，工业设计中常用，用于制造各种机床床身，底座，箱体等。”
  • 球墨铸铁:性能优于灰铸铁，接近于钢。在发动机曲轴，齿轮，活塞等高级铸件中有应用。

1-4 常用碳钢和铝合金

  • Q235/A3
    非标中最常用的一种金属材料，属于普遍碳素结构钢，低碳钢，用于一般的，没什么受力要求的结构件。比如支撑板，底板，机架，工作台面等。也就是非关键结构件很多时候都是用它，便宜好加工。Q表示屈服强度，235 就是指屈服强度为235MpaQ235A，Q235B，Q235C，Q235D，后面的ABCD是质量等级，A最差，D最好其实最主要区别是碳和硫的含量要求不一样,等级越高,含碳量和含硫量会低些所表现出来的性能会好些，一般市场上Q235B最多，且Q235的焊接性要好于Q235A。

  • 45#
    对应日标S45C在金属材料中，非标设计中几乎是最常用的第二种材料。属于优质的碳素结构钢，中碳钢。综合性能优良。故一般零件，大板，结构件，齿轮，链轮，轴等各类结构件都可以用到它，如果是一般的受力要求下几乎是通用的存在。从热处理方面考虑，它属于典型的调质钢，在对耐硬度又要求的场合，可以用它调质处理后使用,有关热处理我们后面会有专门的章节讲解。类似Q235，这种钢材同样便宜，好加工很多设备甚至非关键区域统一使用的此钢。屈服强度490MPA抗拉686而25服235MPA抗拉400MPA左右

第四章 文本注释与块

4-1 单行文字的创建与编辑

  • 命令：TEXT，快捷键 DT
  • 作用：创建一行或多行独立文本对象，适用于简短文字标注。
  • 特点：操作简单、位置精确、编辑灵活。
  • 操作步骤：
    1. 激活命令。
    2. 选择对正方式。
    3. 指定文字起点。
    4. 指定文字高度。
    5. 指定旋转角度。
    6. 输入文本，按两次回车完成。
  • 编辑方法：
    • 移动插入点调整位置。
    • 双击文字修改内容。
    • 选中后按 Ctrl+1 打开特性面板，修改颜色、大小、对正、倾斜等。

4-2 文字的15种对正方式

  • 概念：指定文字插入点（对正点）相对于文字边框的位置。
  • 对正类型：左上(TL)、中上(TC)、右上(TR)、左中(ML)、正中(MC)、右中(MR)、左对齐(L)、居中(C)、右对齐(R)、左下(BL)、中下(BC)、右下(BR)、中间(M)、对齐(A)、布满(F)。
  • 参照线说明：
    • 顶线：大写字母和中文最高点，TL/TC/TR的参照。
    • 基线：大写字母和中文下边线，L/C/R的参照。
    • 中线：基线与顶线的中心线，ML/MC/MR的参照。
    • 底线：小写字母最低点，BL/BC/BR的参照。
  • 特殊对正：
    • **对齐(A)**：文字在两点间平均分布，宽高比不变。
    • **布满(F)**：文字在两点间平均分布，高度不变。
    • **中间(M)**：插入点在顶线与底线中间（几何中心）。

4-3 文字样式的参数

  • 命令：STYLE，快捷键 ST
  • 作用：控制文字外观（字体、字号、效果等）。
  • 关键参数：
    • 字体：系统字体(.ttf)或CAD字体(.shx)。使用.shx字体输入汉字需勾选“使用大字体”。
    • 高度：设置默认文字高度。设为0则在创建文字时单独指定。
    • 效果：颠倒、反向、垂直、宽度因子、倾斜角度。
  • **机械制图标准 (GB/T14665-2012)**：
    • 字体：正体输出。
    • 常用字体：gbenor.shx + gbcbig.shx（正体）；gbeito.shx + gbcbig.shx（斜体）；gdt.shx（形位公差符号）。
  • 外来字体安装：复制.ttf文件到系统“字体”文件夹，重启CAD。

4-4 自定义文字样式

  • 创建步骤：
    1. 输入 ST 打开文字样式窗口。
    2. 点击“新建”，输入样式名。
    3. 定义字体、高度、宽度因子等效果。
    4. 点击“应用”，新样式自动置为当前。
  • 重命名与删除：在样式列表中右键选择。

4-5 多行文字的创建

  • 命令：MTEXT，快捷键 MT 或 T
  • 作用：创建多行段落文字，适用于复杂注释（技术说明、材料清单等）。
  • 特点：格式丰富（加粗、倾斜、颜色、背景等）、支持段落控制（对齐、行距、编号）。
  • 操作步骤：
    1. 激活命令，指定文本框两个角点。
    2. 在文本编辑器中设置全局参数（字体、大小、对正）。
    3. 输入或粘贴文本，用回车换行。
    4. 编辑局部格式，完成后点击“√”或在框外单击。
  • 与单行文字互转：
    • 单行转多行：命令 TXT2MTXT。
    • 多行转单行：使用分解命令 X。

4-6 特殊符号的输入方法

  • 方法一（代码）：%%d→°、%%p→±、%%c→Φ。
  • 方法二（编辑器）：在文字编辑器“插入”面板选择符号。
  • 方法三（右键）：在编辑器中右键选择“符号”。
  • 方法四（堆叠与上下标）：
    • 1/2 → 堆叠 → 正分数。
    • 2#3 → 堆叠 → 斜分数。
    • +0.02^-0.01 → 堆叠 → 极限公差。
    • X^3 → 选中^3 → 堆叠 → 上标。
    • X_2 → 选中_2 → 堆叠 → 下标。
  • 方法五（专用字体）：使用 gdt.shx 字体，输入小写字母可得特定符号（如 n→Ø，o→□）。
  • 方法六（输入法）：使用软键盘输入希腊字母、数学符号等。

4-11 插入表格的三种方法

  • 命令：TABLE，快捷键 TB
  • 表格组成：标题、表头、数据（可简化）。
  • 方法一：从插入点插入
    1. 激活命令，选择“从空表格开始”。
    2. 设置列数/宽、行数/高、单元格式。
    3. 指定插入点，输入文字。
  • 方法二：从指定窗口插入
    1. 激活命令，插入方式选“指定窗口”。
    2. 设置列数和行数。
    3. 在绘图区拖拽窗口确定表格大小，输入文字。
  • 方法三：自数据链接插入（链接Excel）
    1. 激活命令，选择“自数据链接”。
    2. 链接Excel文件，指定插入点。
    3. 注意：修改需解锁Excel源文件，在CAD中“从源下载”同步。

4-12 表格的编辑（上）

  • 编辑文字：双击单元格。
  • 夹点快速编辑：
    • 单击表格边线：拖动角点调整整体大小，拖动边线调整列宽/行高。
    • 选中单元格：拖动夹点调整列宽/行高，右下角青色夹点可自动填充。
  • 表格单元编辑器（选中单元格后）：
    • 插入/删除行或列。
    • 合并/取消合并单元格。
    • 调整对正方式、管理单元样式、添加背景色。
    • 编辑边框（线宽、颜色、双线）。
    • 锁定内容或格式。
    • 设置数据格式（如小数精度、前缀）。

4-14 表格样式的创建与管理

  • 从现有表格创建：在新建时选择现有表格作为基础。

4-17 内部块的创建与插入

  • 块的优势：提高效率、节省空间、便于修改、可添加属性、方便重用。
  • 内部块：仅存在于当前文件。
    • 创建命令：BLOCK，快捷键 B
    • 步骤：激活命令 → 输入名称 → 指定基点（拾取点）→ 选择对象 → 设置选项（如允许分解）→ 确定。
  • 插入块：
    • 命令：INSERT，快捷键 I
    • 步骤：激活命令 → 选择块 → 指定插入点 → 设置比例、旋转、是否分解。

4-18 定义块属性

  • 属性：附着于块的文字信息（如编号、参数）。
  • 创建属性块步骤：
    1. 定义属性：命令 ATT DEF (ATT)，设置标记、提示、默认值、文字设置。
    2. 放置属性：将属性标记放到图形合适位置。
    3. 创建块：将属性和图形一起选中，用 B 命令创建为块。
    4. 插入块：用 I 命令插入，根据提示输入属性值。
    5. 编辑属性：双击属性文字或使用 ATTEDIT 命令。

4-19 定义多个块属性及管理属性

  • 管理命令：BATTMAN（块属性管理器）
  • 功能：编辑、删除、调整属性顺序，设置属性是否可见。

4-20 外部块的创建与使用

  • 外部块（永久块）：保存为独立.dwg文件，可跨文件共享。
  • 创建命令：WBLOCK，快捷键 W
  • 操作：
    1. 激活命令，选择源（块/整个图形/对象）。
    2. 指定基点，选择对象。
    3. 输入文件名和保存路径。
  • 使用：通过插入块命令(I)，浏览到保存的外部块文件插入。

4-21 创建嵌套块

  • 概念：一个块的定义中包含其他块。用于管理复杂组件。
  • 优势：简化编辑，便于块替换和重用。
  • 深度：嵌套层数无限制。

4-22 在位编辑块、快速创建块

  • 在位编辑块：
    • 命令：REFEDIT
    • 操作：选中块后右键“在位编辑块”或输入命令，直接修改后保存。
  • 快速创建块：
    • 操作：选中图形，按住右键拖动，松开后选择“粘贴为块”。
  • 重命名块：菜单栏“格式” → “重命名”，选择“块”。

4-23 块编辑器与块管理

  • 块编辑器：
    • 命令：BEDIT，快捷键 BE
    • 功能：全方位编辑块（绘图、添加约束、定义属性、创建动态块等）。
    • 操作：激活命令 → 选择块 → 在专用编辑环境中修改 → 保存并关闭。
  • 块管理：在块编辑器中可保存、另存为、测试块。

4-24 创建表面粗糙度动态块（上）

    • 作用：删除图中未引用的块定义，优化文件。

4-30 外部参照与DWG比较

  • **外部参照 (XREF, XR)**：将外部文件链接到当前图，保持关联性。
  • 优势：保持一致性、减小文件、便于协同、易于更新。
  • 文件格式：DWG, 图像(JPG,PNG), DWF, DGN, PDF, 点云, 协调模型(NWD/NWC)。
  • 管理操作（在外部参照选项板中）：
    • 附着、卸载、重载、拆离、绑定。
    • 类型：附着型（保留嵌套参照）、覆盖型（忽略嵌套参照）。
  • DWG比较：比较两个DWG文件的差异。

4-31 DWG参照的编辑与应用

  • 编辑操作：
    • 调整透明度（参照淡入）。
    • 在位编辑（类似块）。
    • 打开参照源文件。
    • 创建/删除剪裁边界 (XCLIP)。
    • 隐藏边界 (XCLIPFRAME 设为0)。
  • 主要用途：标准化图框、协同设计、控制文件大小、动态关联更新。

4-34 PDF编辑与应用、PDF转DWG

  • 附着PDF：
    • 命令：PDFATTACH
    • 作用：将PDF作为参考底图插入，可用于描图、图纸恢复。
    • 编辑：调整对比度/淡入度、单色显示、创建剪裁边界、隐藏边框(PDFFRAME 设为0)、启用捕捉。
  • PDF导入（转DWG）：
    • 命令：PDFIMPORT
    • 作用：将PDF几何图形直接转换为可编辑的CAD对象，需后续修复。
  • 用途：描图参考、团队协作、检查验证。

4-35 图像编辑与应用、插入OLE对象

  • 附着图像：
    • 命令：IMAGEATTACH
    • 编辑：调整亮度/对比度/淡入度、创建剪裁边界、隐藏边框(IMAGEFRAME 设为0)、设置背景透明度(PNG)。
  • 徒手画命令：SKETCH，可用 SKETCHINC 调整线段长度，用合并(J)转为多段线。
  • 插入OLE对象：
    • 命令：INSERTOBJ
    • 作用：嵌入其他应用程序的数据（如Word文档、Excel表格）。
    • 操作：激活命令 → 选择“由文件创建” → 浏览文件 → 插入。
    • 编辑：可移动、缩放角点调整大小，右键“编辑”打开源程序修改。
