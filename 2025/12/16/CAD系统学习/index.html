CAD系统学习

第二章 基本图形绘制及练习

2-1 直线命令与动态输入

  • 图标：📏
  • 英文命令：LINE
  • 快捷键：L
  • 操作说明：创建一系列连续的直线段，每条线段可单独编辑。
  • 关键操作：
    • 放弃上一步：U
    • 闭合图元：C
    • 退出绘制：空格/回车、ESC
  • 动态输入 (F12)：
    • 先输入长度，按 TAB 键，再输入角度（角度值不能用空格确认）。
    • 水平向右为0度，角度始终显示正数。

2-2 绘图辅助工具-正交与栅格捕捉

  • 正交 (F8)：限制光标在水平或垂直方向移动。
  • 图形栅格 (F7)：显示栅格点阵，间距可自定义。
  • 栅格捕捉 (F9)：使光标按设定间距移动，通常与栅格间距一致。

2-3 绘图辅助工具-极轴追踪

  • 快捷键：F10
  • 作用：沿设定角度（增量角）的倍数方向进行追踪。
  • 增量角设置：取图形中所有角度的最大公约数。
  • 注意：正交模式与极轴追踪不能同时开启。

2-4 极轴追踪实例练习

  • （练习内容：使用增量角10°进行绘图）

2-5 绝对坐标绘制直线

  • 概念：基于UCS原点 (0,0) 的坐标。
  • 输入格式：#X,Y。开启动态输入时，第一点可省略 #。

2-6 相对坐标绘制直线

  • 概念：相对于前一个输入点的坐标。
  • 输入格式：@X,Y。

2-7 极坐标绘制直线

  • 概念：基于与前一点的距离和角度。
  • 输入格式：@距离<角度。
  • 角度方向：逆时针为正，顺时针为负。

2-8 绘图辅助工具-对象捕捉

  • 快捷键：F3
  • 作用：精确定位到图形的特征点（端点、中点、圆心等）。
  • 临时捕捉：Shift + 右键单击。

2-9 两点之间的中点捕捉

  • 操作：在对象捕捉设置中勾选“端点”捕捉，依次捕捉两个端点即可自动获得中点。

2-10 对象捕捉练习

  • 练习1（平行捕捉）：仅勾选“端点”和“平行线”捕捉，关闭极轴和追踪。
  • 练习2（综合捕捉）：勾选端点、中点、圆心、几何中心、切点、垂足、延长线。

2-11 绘图辅助工具-对象捕捉追踪

  • 快捷键：F11
  • 作用：以捕捉到的点为参考，沿极轴追踪路径定位新点。
  • 两种模式：
    1. 仅正交追踪
    2. 用所有极轴角设置追踪

2-13 正交对齐与极轴角对齐

  • 对齐点：通过两条追踪路径（极轴追踪+对象捕捉追踪）的交点获得。
  • 分类：
    • 正交对齐（两条路径均为正交方向）
    • 单极轴角对齐（一条正交，一条极轴角）
    • 双极轴角对齐（两条均为极轴角方向）

2-15 常用绘图辅助工具的对比

工具 特点 操作方法
对象捕捉 有捕捉标记，无追踪路径 直接单击捕捉特定点
极轴追踪 有追踪路径，与前一点相连 沿增量角倍数方向确定下一点
对象捕捉追踪 有追踪路径，与前一点不相连 捕捉参考点后沿路径确定新点
对齐点获取 有两条相交追踪路径 通过两个方向路径交点确定点

    3. 两点 (2P)
    4. 三点 (3P)
    5. 相切、相切、半径 (T)
    6. 相切、相切、相切

2-25 圆弧的十一种画法

  • 命令：ARC，快捷键 A
  • 常用方法（命令栏快捷操作）：

2-26 优弧与劣弧

  • 定义：包含角小于180°为劣弧，大于180°为优弧。
  • 控制方法：
    • 起点、端点、角度：正值绘劣弧，负值顺时针绘优弧。
    • 起点、端点、半径：正值绘劣弧，负值绘优弧。
    • 按 Ctrl 键可动态切换。

2-33 编辑多段线（下）

  • 命令：PEDIT，快捷键 PE（或双击多段线）
  • 主要功能：

2-34 用角点绘制矩形

  • 命令：RECTANG，快捷键 REC
  • 参数选项：
    • C：倒角
    • F：圆角
    • W：线宽
    • T：厚度
    • E：标高
  • 指定另一角点方式：
    1. 相对坐标：@X,Y
    2. 动态输入：X值，TAB，Y值，回车

2-35 用面积/尺寸/旋转绘制矩形

  • 面积 (A)：先输入面积，再指定长度或宽度。
  • 尺寸 (D)：先输入长度和宽度，再指定方向。
  • 旋转 (R)：指定角度，绘制斜矩形。

2-36 正多边形画法及练习

  • 命令：POLYGON，快捷键 POL
  • 创建步骤：
    1. 输入边数
    2. 指定中心点或选择“边(E)”模式
    3. 若指定中心点，则选择内接于圆(I)或外切于圆(C)
    4. 输入圆的半径

2-38 椭圆弧画法及练习

  • 命令：ELLIPSE，快捷键 EL
  • 画法：

    • 轴，端点：指定一条轴的两个端点→另一条半轴长度
    • 椭圆弧 (A): 先定义椭圆，再指定起始和终止角度
    • 旋转 (R): 通过绕长轴旋转的角度定义椭圆（用于轴测圆）

3-5 延伸命令

  • 命令：EXTEND，快捷键 EX
  • 作用：延伸对象到其他对象的边。
  • 与修剪关系：按 Shift 键可切换延伸与修剪。
  • 两种模式：
    • 快速模式：所有对象自动作为边界边，直接选择要延伸的对象。
    • 标准模式：需先选择边界边，支持多点栏选和边延伸控制。

3-6 延伸练习及锥度斜度画法

  • 锥度：底面直径与高度之比（1:n），如1:5表示每5单位高度直径变化1单位。
  • 斜度：两直线间夹角的正切值（1:n），如1:5表示每5单位水平距离高度变化1单位。
  • 绘图方法：先用相对坐标画一小段，再延伸。

3-7 倒圆角命令

  • 作用：在两条图线间创建指定半径的相切圆弧连接。

    • 半径(R)：定义圆角大小。
    • 多个(M)：连续倒圆角。
    • 修剪(T)：控制源线条是否保留。
    • 多段线(P)：对多段线批量倒圆角。
  • 技巧：按住 Shift 键可恢复角点。

3-26 分解命令及专项练习

3-30 打断命令及操作技巧

  • 命令：BREAK，快捷键 BR
  • 作用：将选定对象打断并删除两点之间的图线（有间隙）。
  • 操作步骤：激活命令 → 选择对象（鼠标落点为第一点）→ 指定第二点。

    1. 第二点选择端点或超出端点，可删除端部。
    2. 将圆打断为圆弧时，删除部分按逆时针方向选择（与点选顺序有关）。

3-39 图案填充的方法及参数设置（上）

  • 命令：HATCH，快捷键 H
  • 作用：使用填充图案对封闭区域进行填充（常用于剖面表示）。
  • 操作步骤：激活命令 → 选择填充区域 → 定义参数 → 确认。
  • 区域选择方法：拾取内部点、选择对象、绘制。

    • 关联：填充是否随边界变化。

3-40 图案填充的参数设置（下）

  • 边界允许间隙：值大于实际间隙一半时仍可填充。
  • 独立的图案填充：控制多条边界时创建单个或多个独立填充。
  • 孤岛检测：
    • 普通：从外向内间隔填充。
    • 外部：仅填充最外层区域。
    • 忽略：忽略所有内部孤岛，全部填充。
  • 放置顺序：后置（在所有对象之后）或前置（在所有对象之前）。

  • 用途：创建封闭多段线或面域，便于填充、计算面积周长、布尔运算等。

  • 夹点编辑：

第四章 文本注释与块

4-1 单行文字的创建与编辑

  • 命令：TEXT，快捷键 DT
  • 作用：创建一行或多行独立文本对象，适用于简短文字标注。
  • 特点：操作简单、位置精确、编辑灵活。
  • 操作步骤：
    1. 激活命令。
    2. 选择对正方式。
    3. 指定文字起点。
    4. 指定文字高度。
    5. 指定旋转角度。
    6. 输入文本，按两次回车完成。
  • 编辑方法：
    • 移动插入点调整位置。
    • 双击文字修改内容。
    • 选中后按 Ctrl+1 打开特性面板，修改颜色、大小、对正、倾斜等。

4-2 文字的15种对正方式

  • 概念：指定文字插入点（对正点）相对于文字边框的位置。
  • 对正类型：左上(TL)、中上(TC)、右上(TR)、左中(ML)、正中(MC)、右中(MR)、左对齐(L)、居中(C)、右对齐(R)、左下(BL)、中下(BC)、右下(BR)、中间(M)、对齐(A)、布满(F)。
  • 参照线说明：
    • 顶线：大写字母和中文最高点，TL/TC/TR的参照。
    • 基线：大写字母和中文下边线，L/C/R的参照。
    • 中线：基线与顶线的中心线，ML/MC/MR的参照。
    • 底线：小写字母最低点，BL/BC/BR的参照。
  • 特殊对正：
    • **对齐(A)**：文字在两点间平均分布，宽高比不变。
    • **布满(F)**：文字在两点间平均分布，高度不变。
    • **中间(M)**：插入点在顶线与底线中间（几何中心）。

4-3 文字样式的参数

  • 命令：STYLE，快捷键 ST
  • 作用：控制文字外观（字体、字号、效果等）。
  • 关键参数：
    • 字体：系统字体(.ttf)或CAD字体(.shx)。使用.shx字体输入汉字需勾选“使用大字体”。
    • 高度：设置默认文字高度。设为0则在创建文字时单独指定。
    • 效果：颠倒、反向、垂直、宽度因子、倾斜角度。
  • **机械制图标准 (GB/T14665-2012)**：
    • 字体：正体输出。
    • 常用字体：gbenor.shx + gbcbig.shx（正体）；gbeito.shx + gbcbig.shx（斜体）；gdt.shx（形位公差符号）。
  • 外来字体安装：复制.ttf文件到系统“字体”文件夹，重启CAD。

4-4 自定义文字样式

  • 创建步骤：
    1. 输入 ST 打开文字样式窗口。
    2. 点击“新建”，输入样式名。
    3. 定义字体、高度、宽度因子等效果。
    4. 点击“应用”，新样式自动置为当前。
  • 重命名与删除：在样式列表中右键选择。

4-5 多行文字的创建

  • 命令：MTEXT，快捷键 MT 或 T
  • 作用：创建多行段落文字，适用于复杂注释（技术说明、材料清单等）。
  • 特点：格式丰富（加粗、倾斜、颜色、背景等）、支持段落控制（对齐、行距、编号）。
  • 操作步骤：
    1. 激活命令，指定文本框两个角点。
    2. 在文本编辑器中设置全局参数（字体、大小、对正）。
    3. 输入或粘贴文本，用回车换行。
    4. 编辑局部格式，完成后点击“√”或在框外单击。
  • 与单行文字互转：
    • 单行转多行：命令 TXT2MTXT。
    • 多行转单行：使用分解命令 X。

4-6 特殊符号的输入方法

  • 方法一（代码）：%%d→°、%%p→±、%%c→Φ。
  • 方法二（编辑器）：在文字编辑器“插入”面板选择符号。
  • 方法三（右键）：在编辑器中右键选择“符号”。
  • 方法四（堆叠与上下标）：
    • 1/2 → 堆叠 → 正分数。
    • 2#3 → 堆叠 → 斜分数。
    • +0.02^-0.01 → 堆叠 → 极限公差。
    • X^3 → 选中^3 → 堆叠 → 上标。
    • X_2 → 选中_2 → 堆叠 → 下标。
  • 方法五（专用字体）：使用 gdt.shx 字体，输入小写字母可得特定符号（如 n→Ø，o→□）。
  • 方法六（输入法）：使用软键盘输入希腊字母、数学符号等。

4-8 多行文字的编辑-样式、格式

4-11 插入表格的三种方法

  • 命令：TABLE，快捷键 TB
  • 表格组成：标题、表头、数据（可简化）。
  • 方法一：从插入点插入
    1. 激活命令，选择“从空表格开始”。
    2. 设置列数/宽、行数/高、单元格式。
    3. 指定插入点，输入文字。
  • 方法二：从指定窗口插入
    1. 激活命令，插入方式选“指定窗口”。
    2. 设置列数和行数。
    3. 在绘图区拖拽窗口确定表格大小，输入文字。
  • 方法三：自数据链接插入（链接Excel）
    1. 激活命令，选择“自数据链接”。
    2. 链接Excel文件，指定插入点。
    3. 注意：修改需解锁Excel源文件，在CAD中“从源下载”同步。

4-12 表格的编辑（上）

    • 插入/删除行或列。
    • 合并/取消合并单元格。
    • 调整对正方式、管理单元样式、添加背景色。
    • 编辑边框（线宽、颜色、双线）。
    • 锁定内容或格式。
    • 设置数据格式（如小数精度、前缀）。

4-17 内部块的创建与插入

  • 块的优势：提高效率、节省空间、便于修改、可添加属性、方便重用。
  • 内部块：仅存在于当前文件。
    • 创建命令：BLOCK，快捷键 B
    • 步骤：激活命令 → 输入名称 → 指定基点（拾取点）→ 选择对象 → 设置选项（如允许分解）→ 确定。
  • 插入块：
    • 命令：INSERT，快捷键 I
    • 步骤：激活命令 → 选择块 → 指定插入点 → 设置比例、旋转、是否分解。

4-18 定义块属性

  • 属性：附着于块的文字信息（如编号、参数）。
  • 创建属性块步骤：
    1. 定义属性：命令 ATT DEF (ATT)，设置标记、提示、默认值、文字设置。
    2. 放置属性：将属性标记放到图形合适位置。
    3. 创建块：将属性和图形一起选中，用 B 命令创建为块。
    4. 插入块：用 I 命令插入，根据提示输入属性值。
    5. 编辑属性：双击属性文字或使用 ATTEDIT 命令。

4-19 定义多个块属性及管理属性

  • 管理命令：BATTMAN（块属性管理器）
  • 功能：编辑、删除、调整属性顺序，设置属性是否可见。

4-20 外部块的创建与使用

  • 外部块（永久块）：保存为独立.dwg文件，可跨文件共享。
  • 创建命令：WBLOCK，快捷键 W
  • 操作：
    1. 激活命令，选择源（块/整个图形/对象）。
    2. 指定基点，选择对象。
    3. 输入文件名和保存路径。
  • 使用：通过插入块命令(I)，浏览到保存的外部块文件插入。

4-21 创建嵌套块

  • 概念：一个块的定义中包含其他块。用于管理复杂组件。
  • 优势：简化编辑，便于块替换和重用。
  • 深度：嵌套层数无限制。

4-22 在位编辑块、快速创建块

  • 在位编辑块：
    • 命令：REFEDIT
    • 操作：选中块后右键“在位编辑块”或输入命令，直接修改后保存。
  • 快速创建块：
    • 操作：选中图形，按住右键拖动，松开后选择“粘贴为块”。
  • 重命名块：菜单栏“格式” → “重命名”，选择“块”。

4-23 块编辑器与块管理

  • 块编辑器：
